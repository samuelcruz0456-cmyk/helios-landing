# HELIOS Landing

Landing page comercial de HELIOS ERP.

HELIOS es un ERP adaptable para empresas que necesitan centralizar ventas,
compras, inventario, recursos humanos, caja, obras, contabilidad, facturacion
y reportes sin cambiar su forma real de operar para encajar en un sistema
generico.

## Publicacion con GitHub Pages

Este sitio es estatico. Puede publicarse directamente con GitHub Pages desde la
rama `main`.

Archivos principales:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `pdf/`
